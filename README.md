# Store Rating System

A full-stack web application where users can view stores, submit ratings, and manage store information.

## Features
- User Authentication
- Store Listings
- Rating System
- Admin Dashboard
- Responsive UI

## Tech Stack
### Frontend
- React.js
- Axios
- CSS / Bootstrap

### Backend
- Node.js
- Express.js

### Database
- MySQL
